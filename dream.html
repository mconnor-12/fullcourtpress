📋 Step 1: Create the Google Sheet (once)

Open sheets.new or use the one you already have. The script creates all tabs automatically — no manual setup needed.

🧪 Step 2: Install the Apps Script

  1. In your Sheet: Extensions → Apps Script
  2. Delete any existing Code.gs content and paste the full script below
  3. Save (Ctrl/Cmd + S) and close the Apps Script editor
  4. Reload the Sheet — you'll see a new 🏀 Full Court Race menu appear (may take 5 seconds). Click it → 🚀 Set up all tabs
  5. Back in Apps Script: Deploy → New deployment → Type: Web app
  6. Execute as: Me · Who has access: Anyone (so anyone with a link can log points)
  7. Copy the /exec URL and paste into the field above
  8. Click Test connection → Push everything to seed the roster

🏀 The Sheet's built-in menu

After installing the script and reloading the Sheet, a 🏀 Full Court Race menu appears at the top. Options:

  • 🚀 Set up all tabs — creates Roster, Totals, Daily Totals, Log, Team Scores, Seating
  • 🔄 Recalc totals now — rebuilds all per-student totals from the Log (useful after editing Log by hand)
  • 📊 Recalc team scores — refreshes the Team Scores summary
  • 📋 Jump to… — quick-navigate to any tab
  • 🔗 Get web app URL — shows your deployed /exec URL in case you lose it
  • ⚠️ Clear all points — wipes Log/Totals/Daily Totals/Team Scores (keeps Roster + Seating)

📊 What gets synced to your Sheet

  • Roster — all 93 students with homeroom + initials
  • Totals — tournament-wide points by student × quarter × category (wide matrix)
  • Daily Totals — one row per student per game day (for charting day-to-day)
  • Log — timestamped append-only audit: date, time, student, quarter, category, points
  • Team Scores — homeroom tournament totals + per-quarter + game days played
  • Seating — current seating layout per homeroom (JSON)

The Race view polls the Sheet every 25 seconds automatically when you have the URL configured, so multiple teachers can log from different browsers and see each other's updates in real time. Manual refresh with ↻ Pull on the Teacher tab anytime.
